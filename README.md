# MaterialDatePicker

=> Single Constraints
- Default Date Picker
- Disable past date, not today date
- Disable future date, not today date
- Disable future date with today date using Calendar.DATE
- Disable weekend date
- Open the date picker with today's date selected.
- *Disable past date with today date using Calendar.DATE
- *Disable past 18 years date with today date using Calendar.DATE

=> Multiple Constraints
- Disable past date, not today and disable weekend date
